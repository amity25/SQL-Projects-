Install mysql workbench from official mysql website here

https://dev.mysql.com/downloads/workbench/

Please look at your Operating system bit and download the same version. 

Installation steps can be seen on youtube and can follow along. 

NOTE - Make sure you don't have an older version of mysql installed before and that you are proceeding with a fresh clean setup. 


STEP 1 — Open MySQL Workbench

When you open it, you will see something like this:

“MySQL Connections”
A box named:
Local instance MySQL80
or something similar

Double click it.

STEP 2 — Enter Password

It may ask:

Enter password for root

Enter the password you created during MySQL installation.

Then click:

OK

STEP 3 — Understand the Screen

After login, you will see:

LEFT SIDE
Navigator
Schemas

This is where databases/tables appear.

CENTER AREA

A big white area.

THIS is where you write SQL queries.

It is called:

SQL Editor

STEP 4 — Write Your First Query

Inside the white editor area, type:

CREATE DATABASE sales_project;

Exactly like this.

STEP 5 — Run the Query

Now VERY IMPORTANT.

At the top toolbar you will see:
⚡ (lightning icon)

Click it.

OR press:

Ctrl + Enter

This runs the query.

STEP 6 — See Output

At the bottom, you will see something like:

Query OK, 1 row affected

That means:
✅ Database created successfully.

STEP 7 — Refresh Left Panel

On left side under:

SCHEMAS

You may NOT immediately see the database.

So:

Right click inside schemas
Click:
Refresh All

OR click the small refresh icon.

Now you should see:

sales_project

🎉 Your first database is created.

VISUAL UNDERSTANDING (Simple Layout)

Your screen roughly looks like this:

------------------------------------------------
| Navigator |  SQL Editor                      |
|            |                                 |
| SCHEMAS    |  CREATE DATABASE sales_project; |
|            |                                 |
|            |                                 |
------------------------------------------------
| Output Area                                  |
| Query OK                                     |
------------------------------------------------

STEP 8 — Start Using Database

Now write this:

USE sales_project;

Run it again using:

Ctrl + Enter

Output:

Database changed

Now MySQL knows:
👉 You want to work inside this database.

STEP 9 — Create Your First Table

Now write:

CREATE TABLE sales (
    order_id INT,
    product_name VARCHAR(50),
    region VARCHAR(50),
    sales_amount INT
);

Run it.

Output:

Query OK

STEP 10 — Refresh Tables

On left side:

sales_project
   → Tables

Refresh schemas.

Now you will see:

sales

🎉 Your first table is ready.

STEP 11 — Insert Data

Now write:

INSERT INTO sales VALUES
(1, 'Laptop', 'North', 50000),
(2, 'Mouse', 'South', 1000),
(3, 'Keyboard', 'East', 2000);

Run it.

STEP 12 — View Your Data

Now write:

SELECT * FROM sales;

Run it.

NOW you will see a proper output table at the bottom.

Example:

order_id	product_name	region	sales_amount
1	Laptop	North	50000
2	Mouse	South	1000
3	Keyboard	East	2000

🎉 CONGRATULATIONS — you just used SQL successfully.

VERY IMPORTANT SHORTCUTS
Run query
Ctrl + Enter
Run all queries
Ctrl + Shift + Enter
